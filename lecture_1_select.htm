Command Stands For
SELECT DISTINCT Inside a table, a column often contains many duplicate values; and sometimes you only want to list the different (distinct) values.
SELECT COUNT(DISTINCT Country) FROM Customers; Returns the count of the distinct country
SELECT COUNT(*) FROM Customers; Returns the count of the all elements
SELECT * FROM Customers WHERE City="Berlin"; Returns the filtered elements
SELECT * FROM Products WHERE Price BETWEEN 50 AND 60; Another example of filtering
SELECT * FROM Customers WHERE Country LIKE 'B%'; This command retuns the elements which starts with B.
SELECT * FROM Customers WHERE City IN ('Paris','London'); This command returns the elements filtered by Paris or London in City column
SELECT * FROM Customers ORDER BY CustomerID DESC; This command returns the ordered data. DESC or ASC keyword determines the sorting direction.
